O que é HTML?

(Voltar ao início)

HTML (HyperText Markup Language) é uma tecnologia utilizada para desenvolver sites. HTML não é uma linguagem de programação.

HTML não é uma linguagem de programação porque não foi projetada para criar algoritmos, manipular variáveis, estruturas de dados, entre outras coisas. Ela é utilizada para estruturar e formatar o conteúdo de páginas web.

O que posso criar com HTML?

(Voltar ao início)

Com o HTML é possível criar vários elementos para construir uma página web com o tema que preferirmos.

Alguns dos elementos que podemos criar são:

  • Títulos e parágrafos;
  • Links e botões;
  • Imagens;
  • Formulários;
  • Vídeos e áudios.
